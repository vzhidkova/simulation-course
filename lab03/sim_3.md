# Клеточные автоматы. Лесные пожары

## Задание:
Реализовать моделирование возникновения и распространения лесных пожаров с использованием двумерного клеточного автомата.

## Состояния клеток:
EMPTY = 0 TREE = 1 FIRE = 2 WATER = 3

## Основные правила:
Горящее дерево на следующем шаге становится пустой
На пустой клетке может вырасти новое дерево с вероятностью 0.01%
Дерево может загореться от удара молнии с вероятностью 0.001%
Дерево загорается, если рядом есть горящий сосед, с вероятностью 50%

## Дополнительные правила:
1. Клетки с водой (на них не может ничего ни вырасти ни загореться)
2. Ветер увеличивает шанс переброса на дерево в сторону ветра увеличивается в зависимости от скорости ветра и соответственно уменьшается в противоположенную (вероятность для остальных 6 клеток не изменяется) Формула для пересчета вероятности $p_{изм} = p_0 +/- v_{ветра}/10$
3. Вероятность загорания на клетках рядом с водой уменьшается вдвое
