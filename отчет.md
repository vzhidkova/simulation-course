# Теоретические и эмпирические характеристики распределения

## Теоретические характеристики

Математическое ожидание:

\[
M[X] = \sum_{i=1}^5 x_i p_i
\]

Дисперсия:

\[
D[X] = \sum_{i=1}^5 (x_i - M[X])^2 p_i
\]

## Эмпирические характеристики

Выборочное среднее:

\[
\overline{X} = \frac{1}{N} \sum_{j=1}^N X_j
\]

Выборочная дисперсия (несмещённая):

\[
S^2 = \frac{1}{N-1} \sum_{j=1}^N (X_j - \overline{X})^2
\]

Эмпирическая вероятность:

\[
P^*(X = i) = \frac{n_i}{N}, \quad n_i - \text{частота значения } i
\]

## Относительные погрешности

Для среднего:

\[
\delta_{\text{mean}} = \left| \frac{\overline{X} - M[X]}{M[X]} \right| \cdot 100\%
\]

Для дисперсии:

\[
\delta_{\text{var}} = \left| \frac{S^2 - D[X]}{D[X]} \right| \cdot 100\%
\]

## Критерий \(\chi^2\) (хи-квадрат)

Статистика критерия:

\[
\chi^2 = \sum_{i=1}^5 \frac{(n_i - Np_i)^2}{Np_i}
\]

где  
- \(n_i\) — наблюдаемая частота значения \(i\),  
- \(Np_i\) — ожидаемая частота.

Число степеней свободы:

\[
df = 5 - 1 = 4
\]

Критическое значение для уровня значимости \(\alpha = 0.05\):

\[
\chi^2_{\text{крит}}(df = 4) = 9.488
\]

Правило принятия гипотезы:

Если \(\chi^2 < \chi^2_{\text{крит}}\), то гипотеза о соответствии распределений принимается, иначе отвергается.
